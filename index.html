CSS Selectors and their Description

Selector Name Description
p,h1 {} Simple Selectors Used to select elements based on name, id, class
div p {} Combinator selectors Select elements based on a specific relationship between them
a:link {} Pseudo-class selectors Select elements based on a certain state
p::first-line {} Pseudo-elements selectors Select and style a part of an element
a[target] {} Attribute selectors Select elements based on an attribute or attribute value
